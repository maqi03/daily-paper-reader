## 摘要
基于评分细则的强化学习（RL）已成为将大语言模型（LLMs）与复杂的开放域指令遵循任务对齐的一种极具前景的方法。然而，现有方法主要依赖于回复级奖励，这引入了严重的奖励稀疏性和奖励歧义问题。为了解决这些问题，我们提出了 Rubrics to Tokens (RTT)，这是一种新型的基于评分细则的强化学习框架，它桥接了粗粒度的回复级评分和细粒度的 Token 级信用分配。RTT 引入了一个 Token 级相关性判别器，用于预测回复中的哪些 Token 负责特定的约束，并通过 RTT-GRPO 优化策略模型，该模型在一个统一的框架内整合了回复级和 Token 级的优势函数。此外，在 Token 级基于评分细则的强化学习中，当从一维的结果级奖励过渡到三维奖励空间时，我们提出了一种名为“样本内 Token 组归一化”（Intra-sample Token Group Normalization）的新型组归一化方法，以适应这一转变。大量的实验和基准测试表明，在不同模型中，RTT 在指令级和评分细则级准确度上均持续优于其他基准方法。

## Abstract
Rubric-based Reinforcement Learning (RL) has emerged as a promising approach for aligning Large Language Models (LLMs) with complex, open-domain instruction following tasks. However, existing methods predominantly rely on response-level rewards, introducing severe reward sparsity and reward ambiguity problems. To address these issues, we propose Rubrics to Tokens (RTT), a novel rubric-based RL framework that bridges coarse response-level scores and fine-grained token-level credit assignment. RTT introduces a Token-Level Relevance Discriminator to predict which tokens in the response are responsible for a specific constraint, and optimizes the policy model via RTT-GRPO, which integrates response-level and token-level advantages within a unified framework. Furthermore, when transitioning from one-dimensional, outcome-level reward to three-dimensional reward space in the token-level rubric-based RL, we propose a novel group normalization method, called Intra-sample Token Group Normalization, to accommodate this shift. Extensive experiments and benchmarks demonstrate that RTT consistently outperforms other baselines in both instruction- and rubric-level accuracy across different models.

### 1. 核心问题与整体含义
*   **研究动机**：现有的指令遵循对齐方法（如基于评分细则的 RL）主要依赖**回复级（Response-level）**奖励。这种粗粒度的反馈存在两个致命缺陷：
    *   **奖励稀疏性**：模型必须满足所有约束才能获得奖励，初学者模型很难通过自我探索获得正向信号。
    *   **奖励歧义性**：两个完全不同的回复可能因为满足了不同数量的约束而获得相同的总分，导致模型无法识别具体是哪些 Token 导致了成功或失败。
*   **核心目标**：将粗粒度的回复级评分细化为 Token 级的信用分配（Credit Assignment），让模型明确知道哪些词满足了约束，哪些词违反了约束。

### 2. 方法论
*   **核心思想**：通过一个专门训练的判别器，将回复级的分数映射到具体的 Token 上，并结合改进的 GRPO 算法进行策略优化。
*   **关键技术细节**：
    *   **Token 级相关性判别器 (Token-Level Relevance Discriminator)**：训练一个小型模型（如 Qwen-4B 或 Llama-3B），输入“指令+约束+回复”，预测每个 Token 与该约束的相关性概率。
    *   **RTT-GRPO 算法**：在 DeepSeek 提出的 GRPO 基础上，将优势函数（Advantage）重构为回复级优势和 Token 级优势的加权和。
    *   **样本内 Token 组归一化 (Intra-sample Token Group Normalization)**：针对三维奖励空间（样本、Token、约束），提出在单个回复内部进行 Token 归一化。这解决了不同回复长度导致的“长度偏差”问题，确保短回复的信号不会被长回复淹没。
*   **算法流程**：策略模型生成一组回复 -> 判别器计算 Token 相关性 -> 结合回复级得分计算 Token 级奖励 -> 进行样本内归一化 -> 更新策略模型。

### 7. 优点
*   **细粒度反馈**：成功解决了 RL 训练中“奖励歧义”的顽疾，提供了更密集的学习信号。
*   **高效性**：在几乎不增加显著算力负担的情况下，大幅提升了训练效率和最终性能。
*   **理论贡献**：识别并定义了多维奖励空间下的“组划分问题（Group Partitioning Problem）”，并给出了数学上的解决方案。

### 8. 不足与局限
*   **判别器依赖**：RTT 的效果高度依赖于 Token 级判别器的准确性，如果判别器本身存在偏见或错误，可能会引导模型走向错误方向。
*   **归一化策略探索有限**：虽然提出了样本内归一化，但对于三维奖励空间中其他可能的组合路径（如先跨约束聚合再跨样本归一化）探索尚浅。
*   **应用场景限制**：目前主要集中在文本指令遵循，尚未在多模态或更复杂的长程 Agent 任务中进行验证。
