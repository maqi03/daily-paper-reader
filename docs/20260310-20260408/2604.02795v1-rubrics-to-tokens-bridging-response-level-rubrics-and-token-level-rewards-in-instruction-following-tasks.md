## 摘要
基于评分细则的强化学习（RL）已成为将大语言模型（LLMs）与复杂的开放域指令遵循任务对齐的一种极具前景的方法。然而，现有方法主要依赖于回答级奖励，这引入了严重的奖励稀疏性和奖励歧义问题。为了解决这些问题，我们提出了 Rubrics to Tokens (RTT)，这是一种新型的基于评分细则的强化学习框架，它桥接了粗粒度的回答级评分和细粒度的 Token 级信用分配。RTT 引入了一个 Token 级相关性判别器，用于预测回答中的哪些 Token 负责特定的约束，并通过 RTT-GRPO 优化策略模型，该模型在一个统一的框架内集成了回答级和 Token 级的优势。此外，在 Token 级基于评分细则的强化学习中，当从一维的结果级奖励过渡到三维奖励空间时，我们提出了一种名为“样本内 Token 组归一化”（Intra-sample Token Group Normalization）的新型组归一化方法，以适应这一转变。大量的实验和基准测试表明，在不同模型中，RTT 在指令级和评分细则级准确率方面均持续优于其他基准方法。

## Abstract
Rubric-based Reinforcement Learning (RL) has emerged as a promising approach for aligning Large Language Models (LLMs) with complex, open-domain instruction following tasks. However, existing methods predominantly rely on response-level rewards, introducing severe reward sparsity and reward ambiguity problems. To address these issues, we propose Rubrics to Tokens (RTT), a novel rubric-based RL framework that bridges coarse response-level scores and fine-grained token-level credit assignment. RTT introduces a Token-Level Relevance Discriminator to predict which tokens in the response are responsible for a specific constraint, and optimizes the policy model via RTT-GRPO, which integrates response-level and token-level advantages within a unified framework. Furthermore, when transitioning from one-dimensional, outcome-level reward to three-dimensional reward space in the token-level rubric-based RL, we propose a novel group normalization method, called Intra-sample Token Group Normalization, to accommodate this shift. Extensive experiments and benchmarks demonstrate that RTT consistently outperforms other baselines in both instruction- and rubric-level accuracy across different models.

### 1. 核心问题与整体含义（研究动机和背景）
论文聚焦于大语言模型（LLM）在处理复杂、多约束的**指令遵循（Instruction Following）**任务时的对齐问题。
*   **研究背景**：现有的强化学习（RL）方法（如基于评分细则的 RL）通常采用“回答级奖励”（Response-level Reward），即根据整个回答是否满足约束给出一个总分。
*   **核心痛点**：
    *   **奖励稀疏性**：如果要求模型同时满足 10 个约束，只要错一个就得 0 分，模型很难通过自我探索获得有效的梯度信号。
    *   **奖励歧义性**：两个完全不同的回答可能因为满足了不同子集的约束而获得相同的总分，导致信用分配（Credit Assignment）模糊，训练不稳定。
*   **核心目标**：将粗粒度的回答级评分细化为 Token 级的奖励信号，明确指出哪些 Token 导致了约束的满足或违反。

### 2. 方法论：核心思想与关键技术
论文提出了 **RTT (Rubrics to Tokens)** 框架，包含三个关键组件：
*   **Token 级相关性判别器 (Token-Level Relevance Discriminator)**：
    *   训练一个辅助模型，输入“指令+约束+回答”，预测回答中每个 Token 与该约束的相关性概率。
    *   通过构建包含“最小修改”和“约束遗漏”的负样本数据集，使判别器能精准定位违规 Token。
*   **RTT-GRPO 算法**：
    *   在 DeepSeek 提出的 GRPO 算法基础上进行扩展。
    *   **综合优势计算**：最终的优势函数（Advantage）由“回答级优势”和“Token 级优势”加权求和组成。
    *   公式逻辑：$A_{sum} = \alpha A_{res} + \beta A_{tok}$。其中 $A_{tok}$ 是通过判别器权重将约束得分映射到具体 Token 上得到的。
*   **样本内 Token 组归一化 (Intra-sample Token Group Normalization)**：
    *   **解决“组划分问题”**：在三维奖励空间（样本、Token、约束）中，传统的跨样本归一化会导致“长度偏差”（长回答主导均值和方差）。
    *   **核心细节**：在单个回答内部对 Token 奖励进行归一化，消除长度干扰，使 Token 级信号与回答级信号解耦。
